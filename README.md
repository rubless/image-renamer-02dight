# 🖼️ Image Renamer 02dight

이미지 파일 이름을 일괄 변경하는 프로그램입니다.

## 기능
- 드래그 앤 드롭으로 이미지 추가
- 매 작업마다 원하는 경로 설정 가능
- 한 번에 여러 장 작업 (생성 시간순 정렬)지원
- 카운트 초기화 버튼으로 다시 01부터 시작 가능

## 원본과 차이점
- 원본은 001의 3자릿수 형식이지만 이 버전은 01의 2자릿수 형식
